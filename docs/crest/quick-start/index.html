安全说明

快速开始

从环境准备到首次登录，再到创建第一个分析页面的最短路径。

本章面向第一次接触 Crest 的用户，帮助你先完成四项基础工作：启动系统、登录系统、认识数据门户和工作台，并创建第一个可查看的分析页面。

适合谁阅读

如仅用于体验系统，可以按本页操作。正式上线前，请继续阅读安装部署、配置管理、安全说明和管理员指南。

准备环境

建议准备一台 Linux 服务器，至少满足以下条件：

项目建议配置
CPU2 核及以上
内存4 GB 及以上
磁盘20 GB 可用空间以上
端口默认开放 8100/tcp
运行时Docker 与 Docker Compose

生产环境建议

正式部署时请使用固定域名和 HTTPS，并把系统访问来源配置为实际域名。生产环境不得使用公开示例密码。

首次登录后建议立即修改管理员密码。生产环境还应尽早完成 HTTPS、备份、组织、角色和权限配置。

登录后如果页面提示账号或密码错误，请回到安装输出确认初始密码。不要尝试猜测默认密码，公开环境也不应长期保留初始密码。

v1.5.6 起统一使用登录入口。普通用户登录后通常进入数据门户，查看已授权的已发布仪表盘和数据大屏；具备后台权限的管理员或编辑人员，可以从门户入口或账号菜单进入后台工作台。

认识数据门户

数据门户用于普通用户查看已发布资源，适合只读访问、部门看板、项目验收和日常经营查看。

门户中常见区域包括：

区域用途
我的收藏固定高频查看的仪表盘和数据大屏
最近使用回到最近打开的资源
全部资源查看当前账号被授权访问的已发布资源
搜索与筛选按关键字、资源类型、更新时间查找资源

门户访问范围由创建人、默认组织、用户授权、组织授权和角色授权共同决定。普通用户只能打开已授权的已发布资源；管理员和系统管理员仍可进入后台进行资源管理。

认识工作台

具备后台访问权限的用户可进入工作台。工作台提供资源概览、快速创建、我的收藏和最近使用。

工作台中的“快速创建”适合用于熟悉产品流程：

  1. 点击“数据源”，接入数据库、Excel 或 API。
  2. 点击“数据集”，选择表、字段或 SQL 构建分析模型。
  3. 点击“仪表盘”或“数据大屏”，把图表组织成展示页面。

第一个分析页面

推荐用下面的路径完成第一次体验：

接入数据源

在“数据准备 / 数据源”中新建数据源，填写连接信息并测试连接。

创建数据集

在“数据准备 / 数据集”中选择数据源表，确认字段后保存数据集。

制作图表

在仪表盘或大屏编辑器中添加图表组件，选择数据集，拖拽维度和指标。

发布页面

预览确认后保存并发布。需要给外部人员查看时，再开启分享链接。

如果目标是制作展示大屏，可以把最后一步换成“新建数据大屏”，在大屏编辑器中添加图表、文字、图片和装饰组件。

快速检查

安装完成后建议检查：

检查项期望结果
登录页能正常打开，Logo 和登录框显示完整
管理员账号使用安装输出的初始密码可以登录
工作台能看到资源统计和快速创建入口
系统设置管理员能进入系统参数、用户管理和权限管理
演示数据新安装环境通常包含演示数据源、数据集和数据大屏
